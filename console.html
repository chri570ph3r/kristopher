In order to display time,javascript's "new Date( )" object is required.

The first step is to define a variable with the above mentioned object.Once this variable is defined,it can be referenced multiple times for different date values,i.e.the year,month,week,date,day,hour,minute and even second!

Second step,you need to create other variables with which to reference the 'new Date( )' object to the corresponding function.

The third step is to create a code that will display the output.This can be done in several ways but for simplicity and briefety,I'll go through the simplest method- 'document.body.innerHTML' In this method,the program overwrites the current function,which in this case is the required time,on the body of the document without requiring prior elements to anchor from.This is not all!

The last step that remains is to specify the refresh rate with which our time display function should rewrite the time after every specified time period.This is done using the 'setInterval( )' method.This function performs a task after every specified interval.
